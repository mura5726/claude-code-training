WHAT IS THIS

「AIで作れる」を、
体感に変える研修。

生成AIの進化で、プログラミングの常識は大きく変わりました。けれど、その実力は資料を読むだけでは伝わりません。

本研修では、エンジニアでない現場のマネージャー・リーダー候補の方にこそ、自分の手で動くアプリを作る体験をしていただきます。「どこまで実務に使えるか」「自部署でどう活かせるか」を、2〜3時間でつかむための入口（Door Opener）研修です。重たい座学ではなく、まず触れてみる場です。

私たち AI Consulting が、保険・人事・不動産などのダッシュボードを Claude Code で実際に構築してきた知見を、そのまま研修に落とし込みました。

Key Outcomes

この研修のゴール

  • AI駆動開発の実力を、自分の目と手で確かめる
  • 自部署の業務への応用アイデアを持ち帰る
  • AIへの「良い頼み方（プロンプト）」を身につける
  • 動くアプリを 1 つ、自分で完成させる
